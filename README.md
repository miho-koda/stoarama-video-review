# Stoarama candidate scanner

Public 150-second review page: https://miho-koda.github.io/stoarama-video-review/review-v2.html

Accepted pilot manifest: [`youtube_review_10.csv`](youtube_review_10.csv)

This tool finds reviewable, contiguous Stoarama video intervals suitable for the
social-mixing pipeline. It does not retain source video: clips are downloaded to
a temporary directory, sampled, and deleted.

An interval passes the default screen when it is at least 150 seconds long,
daylight in at least 80% of samples, contains a person at least 60 pixels tall
in at least 70% of samples, and has more qualifying people than vehicles.

```bash
python scan.py discover --output catalog.csv
python scan.py scan --catalog catalog.csv --output manifest.csv --max-streams 10
python scan.py refresh-links --manifest manifest.csv --output manifest_clickable.csv
```

`manifest.csv` is durable: it stores stream and clip IDs, UTC timestamps, the
Stoarama detail page, and the original source page. `manifest_clickable.csv`
adds short-lived `video_urls` generated by Stoarama. Run `refresh-links` again
whenever those links expire.

YouTube review links follow two rules:

- A continuously running fixed camera may use its ordinary watch URL and leave
  `start_seconds`/`end_seconds` empty.
- A VOD, replay, walking tour, compilation, or mixed day/night video must name a
  passing 150-second interval. Its review URL includes `t=<start_seconds>s`.

Moving and PTZ views are rejected even if a single frame looks suitable. The
scanner checks global frame motion and scene cuts across the whole interval.

Depth Anything V3 is deliberately an optional second-stage review signal. The
existing project wrapper estimates metric depth and intrinsics, but absolute
mounting height also requires fitting a ground plane and recovering camera pose.
Until that estimator is calibrated, `camera_height_status` remains `unscored`.
